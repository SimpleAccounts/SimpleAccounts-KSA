# SimpleAccounts

SimpleAccounts is an innovative accounting software that provides a comprehensive solution for businesses of all sizes. With its user-friendly interface and powerful features, it streamlines financial management and enables businesses to make informed decisions.

## Key Features

- Customer Invoices __*Free__
  - Income Receipts
  - Quotation
- Expense __*Free__
  - Expenses
  - Supplier Invoices
  - Purchase Receipts
- Banking __*Free__
  - Bank Account
- Accountant __*Free__
  - Opening Balance
  - Journals
- Reports __*Free__
  - Financial Reports
    - Profit & Loss
    - Balance Sheet
    - Horizontal Balance Sheet
    - Trial Balances
  - VAT Reports
    - VAT Reports
  - Detailed
    - Detailed General Ledger
  - Expense
    - Expense Details
    - Expense By Category
  - Payrolls
    - Payrolls Summary
- Master __*Free__
  - Chart Of Accounts
  - Contact
  - Product
  - Product Category
  - VAT Category
  - Currency Rate
  - Employee
- Payroll __*Free__
  - Payroll Run
  - Payroll Config
- ZATKA Integration __*Free__
  - e-Invoicing ZATKA Phase-1
  - e-Invoicing ZATKA Phase-2
